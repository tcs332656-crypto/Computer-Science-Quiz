# Computer Science quiz

## Description :

This is a simple python quiz where with the help of lists and functions the computer checks whether the answers provided by user to the questions are correct , then using loops it marks the user based on how many of the ten questions did the user answer correctly, according to the marks obtained the computer provided the user with feedback

## Language used 
-Python

## Features
-Lists/tuples
-if/elif/else statement
-Variables
-Function
-Parameters
-for loops 

## New features I learnt
-How to use functions and lists

## How to run
run the program with :


```bash
python main.py
```
